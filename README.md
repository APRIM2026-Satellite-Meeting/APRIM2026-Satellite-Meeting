# APRIM2026 Satellite Meeting Website

This is a starter website for **APRIM2026 Satellite Meeting**, prepared for deployment with **GitHub Pages**.

## Files
- `index.html` — main homepage
- `styles.css` — site styling

## How to deploy on GitHub Pages
1. Create a new public repository on GitHub.
2. Upload `index.html` and `styles.css` to the repository root.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose:
   - **Source**: Deploy from a branch
   - **Branch**: `main` / `(root)`
5. Save.
6. After a short while, GitHub will publish your site.
